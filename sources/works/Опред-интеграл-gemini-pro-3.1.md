### 1. Вычисление площади плоской фигуры

**Задание:** Найти площадь фигуры, ограниченной графиками функций $y=\sqrt{x+1}$, $y=\sqrt{7-x}$, $y=0$.

**Решение:**
Для начала найдем точки пересечения заданных кривых.

1. Пересечение графиков $y=\sqrt{x+1}$ и $y=\sqrt{7-x}$:
$\sqrt{x+1} = \sqrt{7-x} \Rightarrow x+1 = 7-x \Rightarrow 2x = 6 \Rightarrow x=3$. В этой точке $y=2$.
2. Точки пересечения с осью $y=0$ (ось $Ox$):
* $\sqrt{x+1}=0 \Rightarrow x=-1$.
* $\sqrt{7-x}=0 \Rightarrow x=7$.



Площадь фигуры вычисляется путем интегрирования разности функций на заданных отрезках. Фигура разбивается на две части точкой $x=3$:


$$S = \int_{-1}^{3} \sqrt{x+1}dx + \int_{3}^{7} \sqrt{7-x}dx$$

Вычислим каждый интеграл по отдельности:

* $\int_{-1}^{3} (x+1)^{1/2}dx = \frac{2}{3}(x+1)^{3/2}\Big|_{-1}^{3} = \frac{2}{3}(4^{3/2} - 0) = \frac{2}{3}(8) = \frac{16}{3}$
* $\int_{3}^{7} (7-x)^{1/2}dx = -\frac{2}{3}(7-x)^{3/2}\Big|_{3}^{7} = -\frac{2}{3}(0 - 4^{3/2}) = \frac{16}{3}$

Итоговая площадь:


$$S = \frac{16}{3} + \frac{16}{3} = \frac{32}{3}$$

### 2. Площадь фигуры в полярных координатах

**Задание:** Найти площадь фигуры, ограниченной линиями $r=2\cos 3\varphi$, $r \ge 1$.

**Решение:**
Кривая $r=2\cos 3\varphi$ представляет собой трехлепестковую розу. Условие $r \ge 1$ означает, что нас интересуют части лепестков, находящиеся вне окружности единичного радиуса.
Площадь криволинейного сектора находится по формуле $S = \frac{1}{2}\int_{\alpha}^{\beta} r^{2}(\varphi)d\varphi$. Для площади между двумя кривыми формула имеет вид $S = \frac{1}{2}\int_{\alpha}^{\beta} (r_{2}^{2} - r_{1}^{2})d\varphi$.

Найдем точки пересечения окружности и розы:


$$2\cos 3\varphi = 1 \Rightarrow \cos 3\varphi = \frac{1}{2} \Rightarrow 3\varphi = \pm\frac{\pi}{3} + 2\pi k \Rightarrow \varphi = \pm\frac{\pi}{9} + \frac{2\pi k}{3}$$


Для одного (правого) лепестка пределы интегрирования: $\varphi \in \left[-\frac{\pi}{9}, \frac{\pi}{9}\right]$. В силу симметрии кривой, вычислим площадь половины лепестка (от $0$ до $\frac{\pi}{9}$) и умножим на **2**, а затем на **3** (так как всего лепестков три):


$$S = 3 \cdot 2 \cdot \frac{1}{2} \int_{0}^{\pi/9} \left((2\cos 3\varphi)^2 - 1^2\right) d\varphi = 3 \int_{0}^{\pi/9} (4\cos^2 3\varphi - 1) d\varphi$$

Применим формулу понижения степени: $4\cos^2 3\varphi = 2(1+\cos 6\varphi)$.


$$S = 3 \int_{0}^{\pi/9} (2 + 2\cos 6\varphi - 1) d\varphi = 3 \int_{0}^{\pi/9} (1 + 2\cos 6\varphi) d\varphi$$

$$S = 3 \left[\varphi + \frac{1}{3}\sin 6\varphi\right]_{0}^{\pi/9} = 3 \left(\frac{\pi}{9} + \frac{1}{3}\sin\frac{2\pi}{3}\right) = 3\left(\frac{\pi}{9} + \frac{\sqrt{3}}{6}\right) = \frac{\pi}{3} + \frac{\sqrt{3}}{2}$$

### 3. Объем тела вращения

**Задание:** Найти объем тела, образованного вращением вокруг оси $Ox$ фигуры, ограниченной $x^2/9 - y^2/25 = 1$ и $x^2/25 - y^2/9 = -1$.

**Решение:**
Определим уравнения кривых относительно $y$:

1. $y_1^2 = 25\left(\frac{x^2}{9} - 1\right) = \frac{25}{9}x^2 - 25$
2. $y_2^2 = 9\left(\frac{x^2}{25} + 1\right) = \frac{9}{25}x^2 + 9$

Найдем точки пересечения этих двух гипербол:


$$\frac{25}{9}x^2 - 25 = \frac{9}{25}x^2 + 9 \Rightarrow \left(\frac{25}{9} - \frac{9}{25}\right)x^2 = 34 \Rightarrow \frac{544}{225}x^2 = 34 \Rightarrow x^2 = \frac{225}{16} \Rightarrow x = \pm\frac{15}{4}$$


Фигура состоит из двух симметричных областей (левой и правой), так как первая гипербола не существует на промежутке $x \in (-3, 3)$. Вычислим объем для правой области $x \in \left[3, \frac{15}{4}\right]$ и умножим на **2**.
Объем тела вращения вокруг оси $Ox$ находится по формуле $V_{ox} = \pi \int_{a}^{b} y^{2}(x)dx$.


$$V_{right} = \pi \int_{3}^{15/4} (y_2^2 - y_1^2) dx = \pi \int_{3}^{15/4} \left(34 - \frac{544}{225}x^2\right) dx$$

$$V_{right} = \pi \left[ 34x - \frac{544}{225}\frac{x^3}{3} \right]_{3}^{15/4} = \pi \left[ 34x - \frac{544}{675}x^3 \right]_{3}^{15/4}$$

* Подставляем верхний предел $\frac{15}{4}$:
$34\left(\frac{15}{4}\right) - \frac{544}{675}\left(\frac{3375}{64}\right) = \frac{255}{2} - \frac{8.5 \cdot 5}{1} = 127.5 - 42.5 = 85$
* Подставляем нижний предел **3**:
$34(3) - \frac{544}{675}(27) = 102 - \frac{544}{25} = 102 - 21.76 = 80.24$

$V_{right} = \pi(85 - 80.24) = 4.76\pi = \frac{119}{25}\pi$. Общий объем тела:


$$V = 2 \cdot V_{right} = \frac{238}{25}\pi$$

### 4. Вычисление длины дуги кривой

#### а) В прямоугольной системе координат

**Задание:** $y=\ln(x^2-1)$ на отрезке $2 \le x \le 5$.

**Решение:**
Длина дуги вычисляется по формуле $L = \int_{a}^{b} \sqrt{1+(f^{\prime}(x))^{2}}dx$.
Найдем производную: $y' = \frac{2x}{x^2-1}$.
Тогда $1 + (y')^2 = 1 + \frac{4x^2}{(x^2-1)^2} = \frac{(x^2-1)^2 + 4x^2}{(x^2-1)^2} = \frac{(x^2+1)^2}{(x^2-1)^2}$.


$$L = \int_{2}^{5} \frac{x^2+1}{x^2-1} dx = \int_{2}^{5} \left(1 + \frac{2}{x^2-1}\right) dx = \int_{2}^{5} \left(1 + \frac{1}{x-1} - \frac{1}{x+1}\right) dx$$

$$L = \left[ x + \ln|x-1| - \ln|x+1| \right]_{2}^{5} = \left[ x + \ln\left|\frac{x-1}{x+1}\right| \right]_{2}^{5}$$

$$L = \left(5 + \ln\frac{4}{6}\right) - \left(2 + \ln\frac{1}{3}\right) = 3 + \ln\frac{2}{3} - \ln\frac{1}{3} = 3 + \ln 2$$

#### б) Заданной параметрически

**Задание:** $x=2\cos^3 t$, $y=\sin^3 t$, $0 \le t \le \frac{\pi}{6}$.

**Решение:**
Используем формулу $L = \int_{\alpha}^{\beta} \sqrt{(x^{\prime}(t))^{2}+(y^{\prime}(t))^{2}}dt$.

* $x' = -6\cos^2 t \sin t$
* $y' = 3\sin^2 t \cos t$

$(x')^2 + (y')^2 = 36\cos^4 t \sin^2 t + 9\sin^4 t \cos^2 t = 9\cos^2 t \sin^2 t (4\cos^2 t + \sin^2 t) = 9\cos^2 t \sin^2 t (3\cos^2 t + 1)$.


$$L = \int_{0}^{\pi/6} 3\cos t \sin t \sqrt{3\cos^2 t + 1} dt$$


Введем замену $u = 3\cos^2 t + 1$. Тогда $du = -6\cos t \sin t dt \Rightarrow 3\cos t \sin t dt = -\frac{du}{2}$.
Пределы интегрирования: при $t=0 \Rightarrow u=4$; при $t=\frac{\pi}{6} \Rightarrow u = 3\left(\frac{3}{4}\right) + 1 = \frac{13}{4}$.


$$L = \int_{4}^{13/4} -\frac{1}{2} \sqrt{u} du = \frac{1}{2} \int_{13/4}^{4} u^{1/2} du = \frac{1}{2} \left[ \frac{2}{3}u^{3/2} \right]_{13/4}^{4} = \frac{1}{3} \left( 4^{3/2} - \left(\frac{13}{4}\right)^{3/2} \right) = \frac{8}{3} - \frac{13\sqrt{13}}{24}$$

### 5. Вычисление площади поверхности вращения

**Задание:** Вычислить площадь поверхности, образованной вращением вокруг $Ox$ кривой $x=2\cos t-\cos 2t$, $y=2\sin t-\sin 2t$, $0 \le t \le 2\pi$.

**Решение:**
Так как кривая симметрична относительно $Ox$, полная поверхность вращения генерируется вращением ее верхней половины, где $y \ge 0$ (т.е. на отрезке $0 \le t \le \pi$).
Площадь вычисляется по формуле $S_{ox} = 2\pi \int_{\alpha}^{\beta} y(t)\sqrt{(x^{\prime}(t))^{2}+(y^{\prime}(t))^{2}}dt$.

Найдем производные:

* $x' = -2\sin t + 2\sin 2t$
* $y' = 2\cos t - 2\cos 2t$

Сумма квадратов производных:


$$(x')^2 + (y')^2 = 4\sin^2 t - 8\sin t\sin 2t + 4\sin^2 2t + 4\cos^2 t - 8\cos t\cos 2t + 4\cos^2 2t = 8 - 8\cos(2t-t) = 8(1-\cos t) = 16\sin^2(t/2)$$


Корень из суммы равен $4\sin(t/2)$ (на отрезке $0 \le t \le \pi$ синус положителен).
Представим $y(t)$ через половинные углы:
$y(t) = 2\sin t - \sin 2t = 2\sin t(1 - \cos t) = 2\left(2\sin\frac{t}{2}\cos\frac{t}{2}\right)\left(2\sin^2\frac{t}{2}\right) = 8\sin^3\frac{t}{2}\cos\frac{t}{2}$.

Подставляем в формулу площади поверхности:


$$S_{ox} = 2\pi \int_{0}^{\pi} \left(8\sin^3\frac{t}{2}\cos\frac{t}{2}\right) \left(4\sin\frac{t}{2}\right) dt = 64\pi \int_{0}^{\pi} \sin^4\frac{t}{2}\cos\frac{t}{2} dt$$


Сделаем замену $u = \sin\frac{t}{2}$, $du = \frac{1}{2}\cos\frac{t}{2} dt \Rightarrow \cos\frac{t}{2} dt = 2du$.
Пределы: $t=0 \Rightarrow u=0$; $t=\pi \Rightarrow u=1$.


$$S_{ox} = 64\pi \int_{0}^{1} u^4 (2du) = 128\pi \left[ \frac{u^5}{5} \right]_{0}^{1} = \frac{128\pi}{5}$$

### 6. Несобственные интегралы

**Задание:** Вычислить несобственные интегралы или установить их расходимость.

#### Первый интеграл

$$\int_0^{\infty} e^{-7x}\cos 2x dx$$

Это несобственный интеграл по бесконечному промежутку, который вычисляется как предел $\lim_{b \to +\infty} \int_{a}^{b} f(x)dx$. Найдем первообразную:


$$\int e^{-7x}\cos 2x dx = \frac{e^{-7x}}{(-7)^2 + 2^2} (-7\cos 2x + 2\sin 2x) = \frac{e^{-7x}}{53} (-7\cos 2x + 2\sin 2x)$$


Переходим к пределам:


$$\lim_{b \to \infty} \left[ \frac{e^{-7x}}{53}(-7\cos 2x + 2\sin 2x) \right]_{0}^{b} = 0 - \frac{e^{0}}{53}(-7\cos 0 + 0) = \frac{7}{53}$$


Интеграл сходится и равен $\frac{7}{53}$.

#### Второй интеграл

$$\int_0^1 \cos\left(\frac{\pi}{1-x}\right)\frac{dx}{(1-x)^2}$$

Это несобственный интеграл от функции, имеющей разрыв в точке $x=1$. Введем замену $u = \frac{\pi}{1-x} \Rightarrow du = \frac{\pi}{(1-x)^2}dx \Rightarrow \frac{dx}{(1-x)^2} = \frac{du}{\pi}$.
Пределы интегрирования: при $x=0 \Rightarrow u=\pi$; при $x \to 1^- \Rightarrow u \to +\infty$.


$$\int_{0}^{1} \cos\left(\frac{\pi}{1-x}\right)\frac{dx}{(1-x)^2} = \frac{1}{\pi} \int_{\pi}^{\infty} \cos u du = \frac{1}{\pi} \lim_{b \to \infty} [\sin u]_{\pi}^{b} = \frac{1}{\pi} \lim_{b \to \infty} \sin b$$


Так как предел $\sin b$ на бесконечности не существует, интеграл **расходится**.

### 7. Приложение: Физическая задача

**Задание:** Какую работу требуется выполнить, чтобы с помощью ракеты тело массы $M$ поднять с поверхности Земли (радиус $R$) на высоту $H$.

**Решение:**
Сила гравитационного притяжения меняется в зависимости от расстояния от центра Земли $x$: $F = G\frac{M_{earth} M}{x^2}$.
Известно, что на поверхности Земли (при $x=R$) сила тяжести равна $Mg$, следовательно, $G M_{earth} = g R^2$. Отсюда функция силы равна $F(x) = Mg\frac{R^2}{x^2}$.
Работа, затраченная на преодоление этой силы, вычисляется интегрированием dF по отрезку $[a; b]$, в нашем случае от $x=R$ до $x=R+H$:


$$A = \int_{R}^{R+H} Mg\frac{R^2}{x^2} dx = MgR^2 \int_{R}^{R+H} x^{-2} dx = MgR^2 \left[ -\frac{1}{x} \right]_{R}^{R+H}$$

$$A = MgR^2 \left( \frac{1}{R} - \frac{1}{R+H} \right) = MgR^2 \left( \frac{R+H-R}{R(R+H)} \right) = \frac{MgRH}{R+H}$$
